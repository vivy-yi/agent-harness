# Claw-Eval-Live: A Live Agent Benchmark for Evolving Real-World Workflows

**arXiv ID:** 2604.28139
**Published:** 2026-04-30
**Authors:** Chenxin Li, Zhengyang Tang, Huangxin Lin
**Categories:** cs.AI, cs.CL

## Summary

LLM agents are expected to complete end-to-end units of work across software tools, business services, and local workspaces. Yet many agent benchmarks use static test cases that don't reflect the dynamic nature of real production environments. Claw-Eval-Live introduces a live agent benchmark for evolving real-world workflows.

## Key Contribution

- **Live Benchmark:** Unlike static benchmarks, Claw-Eval-Live evaluates agents against dynamically changing real-world workflows
- **Evolvable Test Cases:** Reflects how production environments change over time
- **Coverage:** Spans software tools, business services, and local workspace contexts
