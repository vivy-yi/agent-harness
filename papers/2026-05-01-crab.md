# Crab: A Semantics-Aware Checkpoint/Restore Runtime for Agent Sandboxes

**arXiv ID:** 2604.28138
**Published:** 2026-04-30
**Authors:** Tianyuan Wu, Chaokun Chang, Lunxi Cao
**Categories:** cs.AI, cs.PL

## Summary

Autonomous agents act through sandboxed containers and microVMs whose state spans filesystems, processes, and runtime artifacts. Crab introduces a semantics-aware checkpoint/restore runtime for agent sandboxes, enabling agents to pause, snapshot, and resume their execution environment with full state preservation.

## Key Contribution

- **Semantics-Aware:** Understands the meaning of sandbox state beyond raw bytes
- **Checkpoint/Restore:** Pause agent execution and resume later from exact state
- **Multi-VM Support:** Works across containers and microVMs
- **State Types:** Filesystems, processes, runtime artifacts

## Relevance to Agent Harness

- Critical infrastructure for long-running agent sessions
- Enables agent state persistence and migration
- Semantics-aware approach vs. raw snapshot — important distinction for agent safety
- Could enable agent "time-travel" debugging and retry mechanisms
