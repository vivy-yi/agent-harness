# Nemobot Games: Crafting Strategic AI Gaming Agents for Interactive Learning with LLMs

- **arXiv:** https://arxiv.org/abs/2604.21896
- **Published:** 2026-04-25
- **Summary:** Nemobot is an interactive agentic engineering environment for creating, customizing, and deploying LLM-powered game agents. Covers dictionary-based, rigorously solvable, heuristic-based, and learning-based games. Uses RLHF, self-critique, and tool-augmented generation.
- **Relevance:** LLM agent engineering paradigm; self-programming AI agent reference
