# Can Coding Agents Reproduce Findings in Computational Materials Science?

## 基本信息
- **论文链接**：http://arxiv.org/abs/2605.00803v1
- **发表日期**：2026-05-01

## 摘要
Large language models are increasingly deployed as autonomous coding agents and have achieved remarkably strong performance on software engineering benchmarks. However, it is unclear whether such success transfers to scientific domains. This paper evaluates whether coding agents can reproduce findings in computational materials science.
