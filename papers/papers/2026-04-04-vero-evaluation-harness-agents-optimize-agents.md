# VeRO: An Evaluation Harness for Agents to Optimize Agents

## 核心贡献

VeRO 是一个评估 harness，支持将 **Agent 优化本身** 作为核心能力来研究。

## 关键亮点

- 专注 coding agents 的自我优化能力
- 提供标准化评估协议
- 支持 Agent-to-Agent 优化场景

## 技术细节

- 主题: Artificial Intelligence (cs.AI)
- 评估框架而非单一 benchmark
- 可扩展到多种 agent 任务
