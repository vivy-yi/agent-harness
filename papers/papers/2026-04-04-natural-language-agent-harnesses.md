# Natural-Language Agent Harnesses (NLAHs)

## 核心贡献

**NLAHs** 将 harness 行为用可编辑的自然语言表达，而非传统代码配置。

## 关键概念

- **NLAH**: Natural-Language Agent Harness，用自然语言描述 harness 规则
- **IHLE**: Intelligent Harness Language Engine，从自然语言自动生成结构化 harness 代码
- 适用于代码合成场景

## 技术细节

- 主题: Computation and Language (cs.CL) + Artificial Intelligence (cs.AI)
- 可编辑、可解释、更易调试
- Agent 可自我修改 harness 行为
