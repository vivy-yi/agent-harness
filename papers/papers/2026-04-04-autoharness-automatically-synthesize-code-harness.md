# AutoHarness: Improving LLM Agents by Automatically Synthesizing a Code Harness

## 核心贡献

AutoHarness 通过自动搜索合成代码 harness，不再需要人工编写 harness 配置。

## 关键亮点

- **Auto Tool Builder**: 自动生成工具代码
- **81% 成功率**: 新工具生成成功率超 81%
- 端到端 harness 自动化

- 主题: Artificial Intelligence (cs.AI)

- 超越人工设计 harness
