**核心内容**:

## 什么是 Agent Harness
Agent Harness 是围绕 AI 模型的基础设施，用于管理长时间运行的任务。实现 Context Engineering 策略：
- Context compaction（上下文压缩）
- Offloading state to storage（状态外置存储）
- Isolating tasks into sub-agents（任务隔离为 sub-agents）

## 为什么需要 Harness
- Benchmark 问题：频繁发布新模型，harness 允许轻松测试对比
- Harness 将模糊的多步骤 agent 工作流转化为可记录、可评分的结构化数据
- 支持 hill-climb 优化
