# Top AI Agentic Workflow Patterns

## 主流编排模式

### 1. Dify — 生产级工作流平台
- 可视化 workflow builder
- 内置 RAG pipeline 管理
- 支持多模型 (OpenAI/Anthropic/开源LLM)
- 本地 + 云端部署

### 2. LangChain — 开发者框架
- Multi-agent 系统
- Tool-using AI agents
- RAG pipelines
- 结构化数据提取

### 3. ReAct — 推理驱动模式
- Reasoning + Acting 交替
- 适合工具调用场景
- 轻量级，易集成

### 4. Multi-Agent Orchestration
- 任务分解到专门 Agent
- 消息总线协调
- 依赖图调度

## 2026 年趋势
- **Skills 化**: Agent 能力封装为可复用技能单元
- **Benchmark 驱动**: Terminal Bench 2.0 等评估基准推动优化
- **Context Engineering**: 最小权限上下文，减少幻觉
