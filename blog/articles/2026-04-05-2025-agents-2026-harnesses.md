# 2025 Was Agents. 2026 Is Agent Harnesses. Here's Why That Changes Everything

**来源**: [Medium - Aakash Gupta](https://aakashgupta.medium.com/2025-was-agents-2026-is-agent-harnesses-heres-why-that-changes-everything-073e9877655e)

**日期**: 2026

**核心论点**:
> "They’re optimizing models when they should be optimizing harnesses."

Claude Code 之所以比同模型的其他产品更好，是因为它的 harness 更好。

**关键案例**:
- **Manus 6个月重写了5次 harness**，模型没变
- Claude Code 的 harness 控制精确的文件系统操作
- Better models don't solve harness problems

**Agent Harness 的职责**:
- 人类审批管理
- 文件系统访问
- 工具编排
- Sub-agent 管理
- Prompt 管理
- 生命周期管理

**2026竞争格局**:
> "竞争优势来自基础设施，而非智能。模型是 commodities，Claude、GPT、Gemini 表现相似。"
