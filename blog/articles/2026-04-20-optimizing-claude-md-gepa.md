**核心内容**:

社区讨论如何优化 AI Agent 指令文件（CLAUDE.md 等 harness 文件），GEPA（Generalized Expert Performance Assessment）方法将 pass rate 从 65% 提升至 85%。

**GEPA 方法要点**:
- 分析 Agent 在实际任务中的失败模式
- 针对失败案例重新设计指令结构
- 引入更精细的评估指标指导指令优化
- 从 65% 到 85%：+20% pass rate 提升

**对 Harness Engineering 的启示**:
- CLAUDE.md / AGENTS.md 等 Agent 指令文件是 Harness 的核心组成部分
- 指令优化是低成本的 Harness 改进手段
- 系统性的评估方法（GEPA）比直觉调优更有效
- 类似的优化可应用于其他 Agent 框架的配置文件
