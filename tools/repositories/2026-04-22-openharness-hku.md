# OpenHarness: Open Agent Harness with Built-in Personal Agent Ohmo

## 概述

**OpenHarness** 是香港大学开源的开放 Agent Harness 项目，特色是内置了个人 Agent "Ohmo"。

## 核心特性

- **开放 Agent Harness**：开源的 Agent 运行框架
- **内置个人 Agent Ohmo**：开箱即用的个人助手
- **可扩展**：支持自定义 Agent 和工具
- **⭐ 10,728**：社区认可度高

## 技术架构

- **Ohmo**：内置的个人 Agent 引擎
- **Harness 框架**：标准化的 Agent 运行环境
- **工具集成**：支持多种外部工具和 API
