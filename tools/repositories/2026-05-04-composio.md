## 描述
TypeScript/Python SDK for AI agents - integrations with Anthropic, OpenAI, LangChain, LlamaIndex, Mastra, MCP, OAuth, multi-provider developer SDK, composable actions, webhook triggers

## 关键特性
- TypeScript + Python双语言SDK
- 多Provider支持（Anthropic, OpenAI, Google Gemini）
- 框架集成（LangChain, LlamaIndex, Mastra, Vercel AI）
- MCP (Model Context Protocol) 支持
- OAuth认证
- 可组合Actions
- Webhook触发器
