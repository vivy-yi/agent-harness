# Loom - AI Harness

## 核心特性

- **Verification Harness**：验证输出质量，确保任务正确完成
- **Structured State**：使用结构化状态而非聊天历史保持模型跟踪
- **Thinking/Acting 路由**：支持在思考模型和行动模型之间路由
- **多接口**：APP / TUI / API / CLI / MCP

## 技术架构

Loom 通过分解任务并通过验证 harness 驱动执行，使 LLM 能够处理复杂任务。
