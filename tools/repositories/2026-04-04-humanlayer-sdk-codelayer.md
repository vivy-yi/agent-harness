# humanlayer

## 概述

HumanLayer 提供 SDK 让 coding agents 在复杂代码库中解决难题时能获得人类干预。

## 核心产品

### HumanLayer SDK
- 在关键节点引入人类判断
- 支持批准/拒绝/修改 agent 行为
- 开源 (Apache 2.0)

### CodeLayer IDE
- AI-first 开发环境
- Battle-tested workflows
- Advanced Context Engineering
- Scale AI-first dev to entire team
