# harness-kit

**仓库**：https://github.com/deepklarity/harness-kit  
**维护**：deepklarity

## 核心理念

> "A kit for building with AI agents — not just the orchestration, but also the engineering patterns around it."

不仅仅是编排，还包括围绕 Agent 构建的工程模式。

## 解决的问题

- 如何让 Agent 的行为更可预测
- 如何构建可靠的验证循环
- 如何设计有效的约束系统
- 如何实现渐进式知识披露

## 适用场景

- 需要构建生产级 Agent 的开发团队
- 关注 Agent 可靠性和安全性的工程师
- 希望学习 Agent 工程最佳实践的学习者

**特点**：实用导向、文档优先、工程化思维
