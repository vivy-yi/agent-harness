# Agent-Harness by Suirotciv

## 核心功能

- **真实世界动作测试**：测试 Agent 在真实环境中的表现
- **开源实现**：开放的测试框架，任何人都可贡献
- **可复现性**：支持测试结果复现和对比

## 测试场景

适合以下 Agent 测试场景：
- 文件系统操作
- API 调用
- 数据库操作
- 网络请求
- 终端命令执行

## 技术栈

基于 Python，与主流 Agent 框架兼容。
