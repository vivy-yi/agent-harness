## 核心定位

Desloppify 是一个 **代码质量 Agent Harness**，专注于将低质量的"slop code"转化为高质量、工程化的代码。

## 为什么值得关注

1. **2691 stars 且持续活跃** — 说明市场对代码质量 agent 有强烈需求
2. **最近更新 (2026-04-05)** — 仍在积极维护
3. **聚焦细分场景** — 不是通用 coding agent，而是专门解决"让烂代码变好"的问题

## Agent Harness 视角

Desloppify 代表了一种 **垂直场景 Agent** 的设计思路：
- 不是让 agent 做所有事
- 而是专门做好一件事：代码质量提升
- 这正是 agent harness 设计的核心思想
