# Condor - AI Trading Agent Harness

## 概述

Condor 是 Hummingbot 项目下的开源框架，用于创建和管理 AI 交易代理。

## 核心功能

- **Agent 创建**：通过 harness 架构快速构建交易代理
- **Agent 管理**：生命周期管理和监控
- **策略集成**：支持多种交易策略

## 适用场景

- 加密货币做市
- 跨交易所套利
- 自动化量化交易
