# agentic-harness by sevenschulte

## 核心概念：六层架构

sevenschulte 提出的 Agentic Coding Harness 六层模型：

| 层级 | 功能 | 说明 |
|------|------|------|
| L1 | 基础执行层 | Shell/终端交互 |
| L2 | 上下文层 | 项目文件、代码结构 |
| L3 | 工具层 | 文件操作、搜索、执行 |
| L4 | 记忆层 | 对话历史、知识积累 |
| L5 | 规划层 | 任务分解、执行计划 |
| L6 | 协作层 | 多 Agent 协调 |

## 支持的 IDE/工具

- **Claude Code**：完整六层配置
