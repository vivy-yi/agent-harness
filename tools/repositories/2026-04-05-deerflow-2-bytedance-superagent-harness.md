# DeerFlow 2.0 - ByteDance Open-Source SuperAgent Harness

**来源**: [GitHub - bytedance/deer-flow](https://github.com/bytedance/deer-flow) | [官网](https://deerflow.tech/)

**发布日期**: 2026年2月28日（v2.0，GitHub Trending #1）

**Stars**: 持续增长中

**核心定位**:
> "Deep Exploration and Efficient Research Flow" — 开源 SuperAgent Harness

**核心架构**:
```
DeerFlow 2.0 = Sub-agents + Memory + Sandboxed Execution + Extensible Skills
```

**主要特性**:
1. **Sub-agents 编排** — 多 agent 协作处理复杂工作流
2. **Memory** — 持久化记忆系统
3. **Sandboxed Execution** — Docker 沙箱安全执行环境
4. **Extensible Skills** — 按需渐进式加载技能
5. **Computer Environment** — 赋予 agent 文件管理、命令执行、长任务运行能力
6. **InfoQuest 集成** — 智能搜索和爬虫工具集

**进化路径**:
从 Deep Research Agent → Full-stack Super Agent

**与 OpenClaw 对比**:
DeepLearning.ai 评价: "DeerFlow 2.0 puts new spin on Claw-like agents" — 两者都强调 agent 编排和 sandbox，但 DeerFlow 侧重研究和深度探索场景。

**安全评估提示**:
> "应被视为 agentic 执行平台，而非普通 AI 助手；需关注执行、数据流、内存、访问和软件摄入的显式控制"
