# nax: AI Coding Agent Orchestrator

## 概述

**nax** 是一个 AI Coding Agent 编排器，核心理念是"持续循环直到完成"（loops until done）。

## 核心特性

- **自主循环执行**：Agent 自动循环执行直到任务完成，无需人工干预
- **LLM 驱动的任务分解**：自动将复杂任务分解为可执行步骤
- **编码专注**：专门针对编程任务的 Agent 编排器
- **开源可扩展**：可与其他 LLM 和工具集成

## 技术架构

```
用户请求 → Nax Orchestrator → LLM Agent Loop → 工具执行 → 结果评估 → 完成/继续
```
