# DeepAgents by LangChain

## 核心功能

- **规划工具**：内置任务分解和规划能力
- **文件系统后端**：完整的文件操作支持
- **子代理生成**：可动态生成子代理处理子任务

## 技术栈

基于 LangChain 和 LangGraph 构建，充分利用其图结构编排能力。
