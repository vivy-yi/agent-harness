# CUGA: Enterprise Generalist Agent Harness

## 概述

**CUGA** 是一个面向企业的开源通用 Agent Harness，支持在网页和 API 上执行复杂任务。

## 核心特性

- **通用 Agent 能力**：支持多种任务的通用 Agent 框架
- **网页/API 任务执行**：自动化执行复杂网页操作和 API 调用
- **OpenAPI/MCP 集成**：支持标准化的 API 集成
- **可组合架构**：模块化设计，灵活扩展
- **推理模式**：内置多种推理策略
- **策略感知**：支持企业策略执行和合规

## 技术亮点

- **MCP (Model Context Protocol)** 支持
- **OpenAPI** 规范集成
- 企业级安全策略引擎
- 多模型支持
