# agent-bridge: 多Agent对话平台

## 概述

**agent-bridge** 是一个多 Agent 对话平台，基于 OpenClaw 和 Hermes 构建，支持多 Agent 辩论与协作。

## 核心特性

- **OpenClaw 集成**：利用 OpenClaw 的 Agent 框架能力
- **Hermes 消息网关**：支持多 Agent 间的高效通信
- **辩论模式**：支持多个 Agent 就特定话题进行辩论
- **协作能力**：多个 Agent 协同完成复杂任务

## 技术架构

- **OpenClaw**：作为 Agent 运行时框架
- **Hermes**：多 Agent 消息网关和路由
- **辩论引擎**：处理多 Agent 对话逻辑
