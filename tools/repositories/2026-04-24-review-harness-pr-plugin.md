# review-harness

## 核心功能

- **PR 审查能力**：教 Agent 如何审查 Pull Requests
- **灵感来源**：借鉴 browser-harness 的设计理念
- **模块化**：可作为其他 harness 的插件使用

## 设计理念

browser-harness 证明了为特定任务设计专用 harness 的价值。review-harness 将这一理念应用到代码审查场景。

## 技术栈

预计基于 TypeScript/Node.js，可与主流 Agent 框架集成。
