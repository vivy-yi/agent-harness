# lobehub: Agent Team Collaboration Platform

## 概述

**lobehub** 是一个多 Agent 协作平台，定位为"工作和生活终极空间"，支持发现、构建和协作具有成长性的 Agent teammates。

## 核心特性

- **多 Agent 协作**：支持多个 Agent 协同工作
- **Agent 团队设计**：轻松设计和配置 Agent 团队
- **Agent 作为工作单元**：以 Agent 作为交互的基本单位
- **可成长**：Agent 能够随使用不断学习和成长
- **大规模社区**：⭐ 75,447 的高星项目

## 技术亮点

- TypeScript 实现
- 支持 OpenClaw Agent 运行时
- 工作流驱动的 Agent 编排
- 丰富的 Agent 生态
